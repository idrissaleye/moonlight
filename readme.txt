1)installer ollama
2)faire "ollama run deepseek-r1:1.5b" dans un terminal ou cmd prompt
3)assurez-vous d'avoir python sur votre machine
4)taper la commande "pip install -r requirements.txt" pour installer les dépendances environnementals de python (vous devez vous trouver dans le répertoire qui contient requirements.txt dabord)
5)installer XAMPP
6)activer mysql et apache dans XAMPP
7)héberger le dossier Moonlight dans htdocs de XAMPP
8)uploader le fichier plateforme_examens.sql dans php_admin textuellement sinon la db ne sera pas reconnu par les fichiers
9)exécuter le fichier ia.py dans le dossier Moonlight et laisser fonctionner en arrière-plan
10)exécuter le fichier pdf.py dans le dossier Moonlight et laisser fonctionner en arrière-plan
11)taper textuellement: "localhost/Moonlight/moonlight.html" et vous devriez attérir sur la page d'accueil 100% fonctionnelle de moonlight
